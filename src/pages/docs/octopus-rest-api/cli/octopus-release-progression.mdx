Manage progression of a release in Octopus Deploy

```
Usage:
  octopus release progression [command]

Available Commands:
  allow Allows a release to progress to the next phase.
  help Help about any command
  prevent Prevents a release from progression to the next phase

Global Flags:
  -h, --help                   Show help for a command
      --no-prompt              Disable prompting in interactive mode
  -f, --output-format string   Specify the output format for a command ("json", "table", or "basic") (default "table")
  -s, --space string           Specify the space for operations


Use "octopus release progression [command] --help" for more information about a command.
```

## Examples

<SamplesInstance />

```
$ octopus release progression prevent
$ octopus release progression allow


```
