List project branches in Octopus Deploy

```
Usage:
  octopus project branch list [flags]

Aliases:
  list, ls

Flags:
      --git-ref string   The git-ref for the Config-As-Code branch
  -p, --project string   The project


Global Flags:
  -h, --help                   Show help for a command
      --no-prompt              Disable prompting in interactive mode
  -f, --output-format string   Specify the output format for a command ("json", "table", or "basic") (default "table")
  -s, --space string           Specify the space for operations

```

## Examples

<SamplesInstance />

```
$ octopus project branch list "Deploy Website"
$ octopus project variable ls


```
