Manage runbook snapshots in Octopus Deploy

```
Usage:
  octopus runbook snapshot [command]

Available Commands:
  help Help about any command
  list List runbook snapshots

Global Flags:
  -h, --help                   Show help for a command
      --no-prompt              Disable prompting in interactive mode
  -f, --output-format string   Specify the output format for a command ("json", "table", or "basic") (default "table")
  -s, --space string           Specify the space for operations


Use "octopus runbook snapshot [command] --help" for more information about a command.
```

## Examples

<SamplesInstance />

```
$ octopus runbook snapshot create
$ octopus runbook snapshot list


```
