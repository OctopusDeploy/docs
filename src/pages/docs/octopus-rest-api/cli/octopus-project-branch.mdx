Manage project branches in Octopus Deploy

```
Usage:
  octopus project branch [command]

Aliases:
  branch, variable

Available Commands:
  create Create a Git branch for a project
  help Help about any command
  list List project branches

Global Flags:
  -h, --help                   Show help for a command
      --no-prompt              Disable prompting in interactive mode
  -f, --output-format string   Specify the output format for a command ("json", "table", or "basic") (default "table")
  -s, --space string           Specify the space for operations


Use "octopus project branch [command] --help" for more information about a command.
```

## Examples

<SamplesInstance />

```
$ octopus project branch list "Deploy Web App"
$ octopus project branch create -p "Deploy Web App" --new-branch-name add-name-variable --base-branch refs/heads/main -


```
