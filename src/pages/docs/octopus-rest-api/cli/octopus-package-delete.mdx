Delete a package in Octopus Deploy

```
Usage:
  octopus package delete {<id>} [flags]

Aliases:
  delete, del, rm, remove

Flags:
  -y, --confirm             Don't ask for confirmation before deleting the package.
  -p, --package-id string   The Package ID of the package to delete
  -v, --version string      The version of the package to delete


Global Flags:
  -h, --help                   Show help for a command
      --no-prompt              Disable prompting in interactive mode
  -f, --output-format string   Specify the output format for a command ("json", "table", or "basic") (default "table")
  -s, --space string           Specify the space for operations

```

## Examples

<SamplesInstance />

```
$ octopus package delete Packages-1
$ octopus package rm Packages-1
$ octopus package del --package-id ThePackage --version 1.0.0


```
