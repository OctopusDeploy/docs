List runbook snapshots in Octopus Deploy

```
Usage:
  octopus runbook snapshot list [flags]

Aliases:
  list, ls

Flags:
      --limit int32      Limit the maximum number of results that will be returned (default 2147483647)
  -p, --project string   Name or ID of the project to list runbook snapshots for
  -r, --runbook string   Name or ID of the runbook to list snapshots for


Global Flags:
  -h, --help                   Show help for a command
      --no-prompt              Disable prompting in interactive mode
  -f, --output-format string   Specify the output format for a command ("json", "table", or "basic") (default "table")
  -s, --space string           Specify the space for operations

```

## Examples

<SamplesInstance />

```
$ octopus runbook snapshot list --project "Deploy Web App" --runbook "Run maintenance"
$ octopus runbook snapshot ls


```
