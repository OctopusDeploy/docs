This script uploads a certificate file in pfx format to the Octopus [certificate library](/docs/deployments/certificates) to be used in deployments or runbooks.

## Usage

Provide values for:

- Octopus URL
- Octopus API Key
- PFX file path
- PFX file password
- Certificate Name
