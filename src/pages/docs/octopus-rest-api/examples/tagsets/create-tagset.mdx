This script demonstrates how to programmatically create a tag set in Octopus Deploy.

:::div{.hint}
**Extended features require version 2025.4.3897+**

The script examples include `Type` and `Scopes` parameters that require Octopus Deploy version **2025.4.3897** or later with the `extended-tag-sets` feature toggle enabled.

**Without extended features:** The API will ignore `Type` and `Scopes` parameters and create tag sets with **MultiSelect** type and **Tenant** scope only.
:::

## Usage

Provide values for:

- Octopus URL
- Octopus API Key
- Name of the space to use
- Name of the tag set to create
- Type for the tag set (MultiSelect, SingleSelect, or FreeText)
- Scopes for the tag set (Tenant, Environment, or both)
- Optional description for the tag set
- Optional tags to add to the new tag set
